# sticky コマンドリスト

操作の実装正（SSOT）は `操作一覧表.md`。本書はひと目で引けるクイックリファレンス。

---

## グローバル（常時）

| 操作 | 動作 |
|---|---|
| `Cmd + Option + Enter` | 1枚セッション新規生成 |
| `Cmd + Option + N` | セッション枚数ピッカーを開く |

---

## ピッカー表示中

| 操作 | 動作 |
|---|---|
| `1` 〜 `9` | 指定枚数のセッションを生成 |
| `Esc` / `⌫` | ピッカーを閉じる |
| その他キー | ブロック（無効） |

---

## 非選択（none）

| 操作 | 動作 |
|---|---|
| メモをクリック | メモを選択 |
| メモをダブルクリック | 編集開始 |
| メモを右クリック | セッション文脈メニューを表示 |
| メモ外クリック | 何もしない |
| `Esc` | 何もしない |

---

## メモ選択中（memo_selected）

| 操作 | 動作 |
|---|---|
| `Enter` | 編集開始 |
| `Esc` | 選択解除 |
| `Cmd + S` | 保存・閉じる |
| `Cmd + Enter` | 保存・選択解除（メモは残る） |
| `⌫` / `Del` | 削除確認モーダル（pin 中は無効） |
| `p` | pin トグル |
| ドラッグ | メモを移動（pin 中は無効） |
| 四隅ハンドルをドラッグ | リサイズ（pin 中は無効） |
| メモ外クリック | 選択解除 |

---

## 編集中（editing）

| 操作 | 動作 |
|---|---|
| `Enter` | 改行 |
| `Esc` | 編集終了・選択解除（メモは残る） |
| `Cmd + S` | 保存・閉じる |
| `Cmd + Enter` | 保存・編集終了（メモは残る） |
| `⌫` / `Del` | テキスト削除 |
| `p` | `p` の文字入力 |
| メモ外クリック | 編集終了・選択解除 |

---

## セッション選択中（session_selected）

| 操作 | 動作 |
|---|---|
| `Esc` | 選択解除 |
| `Cmd + S` | 保存・セッションを閉じる |
| `Cmd + Enter` | 保存・選択解除（セッションは残る） |
| `⌫` / `Del` | セッション削除確認（pin 中は無効） |
| `p` | セッション内全メモを pin スマートトグル |
| セッション内メモをドラッグ | 全メモを一括移動（pin 済みは除外） |
| セッション内メモをクリック | そのメモを個別選択（memo_selected へ） |
| メモ外クリック | 選択解除 |

---

## 削除確認モーダル表示中

| 操作 | 動作 |
|---|---|
| `Enter` / `⌫` / `Del` | 削除確定 |
| `Esc` | キャンセル |
| モーダル外クリック | キャンセル |
| その他キー | ブロック（無効） |

---

## 右クリックメニュー（メモ上）

| 項目 | 動作 |
|---|---|
| このセッションを選択 | session_selected に入る |
| このセッションを閉じる | セッションをデスクトップから閉じる |
| このセッションを削除... | 削除確認モーダルを表示 |

---

## メニューバー（sticky メニュー）

| 項目 | 動作 |
|---|---|
| New 1-Note Session | 1枚セッション新規生成 |
| Open Home | （Phase 5 実装予定） |
| Open Trash | （Phase 5 実装予定） |
| Open Settings | （Phase 5 実装予定） |
| Quit sticky | アプリ終了 |

---

## Pin の挙動まとめ

| 操作 | pin 中の挙動 |
|---|---|
| ドラッグ移動 | 不可 |
| リサイズ | 不可 |
| 削除 | 不可 |
| 編集 | 可能 |
| session drag での移動 | 除外（その場に留まる） |
