# Post it 操作一覧表

## 1. 目的
本書は、デスクトップ上のメモ操作と管理画面上の操作について、状態ごとの入力と挙動を整理するための一覧表である。  
実装時に入力競合や状態不整合を避けることを目的とする。

## 2. デスクトップ操作一覧
### 2.1 共通

| 入力 | 条件 | 動作 |
| --- | --- | --- |
| `Cmd + Option + Enter` | 常時 | 1枚セッションを開始 |
| `Cmd + Option + N` | 常時 | 新規セッション枚数選択UIを開く |
| 枚数選択UI内の `1` 〜 `9` | 新規セッション枚数選択UI表示中 | 指定枚数の新規セッションを作成 |
| 領域外クリック | セッション選択中 / メモ選択中 | 選択解除 |
| `Esc` | セッション選択中 / メモ選択中 | 選択解除 |
| シングルクリック | メモ上 | 対象メモを選択 |
| ダブルクリック | メモ上 | 編集開始 |
| 右クリック | メモ上 | セッション文脈メニュー表示 |
| ドラッグ開始 | クリック位置から `6px` 以上移動 | ドラッグとして扱う |

### 2.2 セッション選択状態

| 入力 | 条件 | 動作 |
| --- | --- | --- |
| `このセッションを選択` | 右クリックメニュー | 対象セッションを選択し最前面化 |
| ドラッグ | セッション選択中 | セッション内メモを一括移動 |
| `Cmd + S` | セッション選択中 | セッション内の保存対象メモを更新保存する |
| `Cmd + Enter` | セッション選択中 | セッションを保存して閉じる |
| `Delete` | セッション選択中 | セッション削除確認モーダル表示 |
| シングルクリック | 同一セッション内メモ上 | そのメモを個別選択 |

### 2.3 メモ選択状態

| 入力 | 条件 | 動作 |
| --- | --- | --- |
| ドラッグ | メモ選択中 | 対象メモを移動 |
| `Enter` | メモ選択中 | 編集開始 |
| ダブルクリック | メモ選択中 | 編集開始 |
| `Cmd + S` | メモ選択中 | 対象メモを更新保存する |
| `Cmd + Enter` | メモ選択中 | 対象メモを保存して閉じる |
| `Delete` | メモ選択中 | メモ削除確認モーダル表示 |

### 2.4 編集状態

| 入力 | 条件 | 動作 |
| --- | --- | --- |
| `Enter` | 編集中 | 改行 |
| `Esc` | 編集中 | 編集終了し選択解除 |
| 範囲外クリック | 編集中 | 編集終了し選択解除 |
| `Cmd + S` | 編集中 | 対象メモを更新保存する |
| `Cmd + Enter` | 編集中 | 保存して閉じる |

### 2.5 削除確認モーダル

| 入力 | 条件 | 動作 |
| --- | --- | --- |
| `Delete` | 削除確認中 | 削除確定してゴミ箱へ移動 |
| `Enter` | 削除確認中 | 削除確定してゴミ箱へ移動 |
| `Esc` | 削除確認中 | モーダルを閉じる |
| 確認ボタン | 削除確認中 | 削除確定 |
| キャンセルボタン | 削除確認中 | キャンセル |

## 3. 自動処理一覧

| トリガー | 条件 | 動作 |
| --- | --- | --- |
| 5分ごとの内部保存 | 開いているセッションあり かつ 未保存変更あり | ローカル更新保存 |
| 自動閉じるタイマー到達 | 非選択・非編集のセッション | 保存してデスクトップから閉じる |
| 自動閉じるタイマー到達 | 選択中または編集中 | 何もしない |
| セッション内メモ数が0 | 常時 | セッションを消滅させる |
| セッション上限超過作成 | 5セッション表示中 | 新規作成せず警告表示 |
| メモ上限超過作成 | 15メモ表示中 | 新規作成せず警告表示 |
| アプリ起動 | 常時 | `is_open` を全リセット後、空メモと空セッションをクリーンアップ |

## 4. 管理画面操作一覧
### 4.1 Home

| 入力 | 条件 | 動作 |
| --- | --- | --- |
| クリック | メモカード上 | メモ選択 |
| Shift+クリック | 管理画面で複数選択対応時 | 複数選択 |
| 右クリック | メモまたはセッション上 | コンテキストメニュー表示 |
| ドラッグ&ドロップ | メモ選択中 | 別セッションへ移動 |
| `Delete` | メモ選択中 | 削除確認 |
| 開く | メモまたは複数メモ選択中 | デスクトップへ再表示 |
| 検索入力 | 検索欄 | タイトル・本文のキーワード一致検索 |

### 4.2 ゴミ箱

| 入力 | 条件 | 動作 |
| --- | --- | --- |
| 復元 | ゴミ箱内メモ/セッション | 元に戻す |
| 完全削除 | ゴミ箱内メモ/セッション | 物理削除 |

### 4.3 設定

| 入力 | 条件 | 動作 |
| --- | --- | --- |
| 自動閉じる時間変更 | 設定画面 | `5 / 10 / 30 / 60分` に更新 |

## 5. 警告表示

| 状態 | 表示 | 文言 |
| --- | --- | --- |
| セッション上限到達 | 赤半透明オーバーレイ + 横揺れ | `これ以上セッションは開けません` |
| セッション上限到達 | 管理画面 | `セッション上限: 5/5` |
| メモ上限到達 | 赤半透明オーバーレイ + 横揺れ | `これ以上メモは表示できません` |
| メモ上限到達 | 管理画面 | `表示メモ上限: 15/15` |

## 6. 実装メモ
- `Cmd + C` はコピーと競合するため閉じる操作に使わない
- セッション移動は `session_id` を移行先へ更新することで実現する
- 削除は即物理削除せず、必ずゴミ箱経由にする
- 自動閉じるは保存とセットで扱う
- デスクトップ上の選択モデルは `メモ選択 -> 編集` を基本とし、セッション選択は右クリックメニュー経由で入る
- 保存処理には明示保存と autosave の両方を含み、どちらも `title` を再生成する
- 使用中 `color_slot` の取得は、`is_open = 1` のセッションだけを対象にする
- スロットは `1〜15` の前詰め方式で扱い、移動されたメモの元スロットは空きとして再利用する
